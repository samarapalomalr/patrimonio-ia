# Patrimônio IA

## 📌 Projeto de Iniciação Científica Tecnológica

**Programa Institucional de Bolsas de Iniciação em Desenvolvimento Tecnológico e Inovação**

### 🎯 Título
Desenvolvimento de sistema inteligente para classificação de elementos construtivos em edificações históricas

---

## 🧠 Descrição do Projeto
Este projeto tem como objetivo o desenvolvimento de um **protótipo de sistema inteligente** para a **classificação automática de elementos construtivos** presentes em edificações históricas, utilizando técnicas de **Inteligência Artificial e Aprendizado Profundo**.

O sistema visa auxiliar a análise, catalogação e preservação do patrimônio histórico por meio de uma solução tecnológica acessível e escalável.

---

## 🎯 Objetivo Geral
Desenvolver um sistema inteligente capaz de classificar elementos construtivos em edificações históricas, integrando algoritmos de aprendizado profundo a uma interface gráfica interativa.

---

## 📚 Objetivos Específicos
- Estudar conceitos fundamentais de:
  - Inteligência Artificial  
  - Aprendizado de Máquina  
  - Engenharia de Software  
  - Construções históricas
- Aplicar técnicas de **Aprendizado Profundo** para classificação de elementos construtivos.
- Desenvolver uma **interface gráfica** e integrar o sistema com os modelos inteligentes.

---

## 🛠️ Tecnologias Utilizadas
- **Flutter** (desenvolvimento multiplataforma)
- **Dart**
- **Firebase** (armazenamento / backend)
- **Aprendizado Profundo** (modelos de classificação)
- **IA aplicada ao patrimônio histórico**

---

## 📱 Plataforma
O sistema é desenvolvido utilizando **Flutter**, permitindo execução em:
- Web
- Android
- iOS
- Desktop (Windows, Linux, macOS)
